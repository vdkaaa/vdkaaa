Featured Projects
🔬 XR Performance Lab

Modular XR performance testing system — Unity 6 / URP

A technical tool for running isolated, reproducible rendering experiments in Unity. Each experiment modifies one variable, holds the configuration while metrics stabilize, then restores state automatically via a guaranteed try/finally teardown.

Interface-driven architecture: IExperiment, IExperimentRunner, IExperimentRegistry
18 experiment instances across 5 categories: MSAA, Render Scale, Shadows, CPU Throttle, Transparency Overdraw
Live metrics overlay: FPS, frame time, batches, triangles, vertices
Target platforms: Meta Quest, PCVR
